## WayPointer

A navigation system using the google maps api and ORtools to calculate the most efficient route between multiple waypoints.

## Features

- Register/Login
- Firebase database for unique user accounts
- Accessable Google Map
- Autofill locations
- Ability to enter multiple waypoints, including a Start and End destination
- Portray route segments individually between waypoints
- Send sms with the current route to multiple devices/users
- Ability to save routes

## How To Use?
It's easy!

1. Register with WayPointer using your email.
2. Log into WayPointer.
3. Enter your destinations and hit Submit!
